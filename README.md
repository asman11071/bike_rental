# bike_rental
🏍️ Bike Rental System
A simple web-based Bike Rental System built using HTML, CSS, PHP, and MySQL (XAMPP).
This project allows users to browse bikes, register/login, and book rentals through a user-friendly interface.

📖 Description
The Bike Rental System is designed to simplify the process of renting bikes online. Users can view available bikes, check pricing, and book them for specific durations.

This project demonstrates fundamental concepts of:

Frontend design

Backend development with PHP

Database integration using MySQL

✨ Features
🔐 User Registration & Login

🏍️ View Available Bikes

📅 Bike Booking System

💰 Rental Price Display

📋 Booking Records

🧾 Simple Dashboard

🛠️ Tech Stack
Frontend: HTML, CSS

Backend: PHP

Database: MySQL

Server: XAMPP

⚙️ Installation & Setup
Install XAMPP

Start Apache and MySQL

Move project folder to:

C:\xampp\htdocs\
Open http://localhost/phpmyadmin/

Create a database (e.g., bike_rental)

Import the .sql file into the database

Run the project:

http://localhost/Bike-Rental-System/
